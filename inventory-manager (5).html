☁️ Cloud Database Sync

Connect this device to your backend server so products, sales, and stock stay in sync across every device — instead of each device only keeping its own local copy. Your existing Backup, Restore, and Live File Sync features still work exactly as before, connected or not.

🗑️ Reset Account

A backup of your current inventory (products, sales, categories) downloads automatically first. After that, this permanently deletes all products, sales, and user accounts on this device. This cannot be undone.

📥 Restore Backup

This backup file contains data.

Merge combines this backup with what's already here — safe, no data lost, and it'll ask you to pick a value for any product where the numbers disagree.

Replace wipes current data and uses only what's in the backup file.

⚖️ Resolve Differences

These products exist on both sides with different values. For each one, you can see exactly what changed, who changed it, and when — the more recently edited version is pre-selected for you.

🧩 Conflict Resolution Center

Everything that could be merged automatically already has been. For each field below, both sides made a deliberate change — you'll see exactly what each person set it to and when, so you know exactly what you're choosing between. Every decision is recorded permanently in the Event Log.
